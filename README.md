# gltf2bb

Convert glTF character model to Blockbench project, optimized for PMX (format conversion required) and VRM

Quality optimization in progress

## Features

- Convert glTF character model to Blockbench project
- Native VRM 1.0 support
## Installation

Requires UV installed

```bash
uv sync
```
